# Font Assets for Landing Page

Place the following font files in this folder:

## Required Fonts:
1. **zentry-regular.woff2** - Main display font for headings
2. **circularweb-book.woff2** - Secondary font
3. **robert-medium.woff2** - Body text font (medium weight)
4. **robert-regular.woff2** - Body text font (regular weight)

## Where to Get Fonts:
You can use Google Fonts or similar fonts as alternatives:
- Zentry → Try "Orbitron" or "Exo 2" (bold/black weight)
- Circular Web → Try "Inter" or "DM Sans"
- Robert → Try "Open Sans" or "Roboto"

## How to Add Fonts:
1. Download font files in WOFF2 format
2. Place them in this `/public/fonts/` directory
3. The fonts are already configured in globals.css

## Convert TTF/OTF to WOFF2:
Use: https://cloudconvert.com/woff2-converter
