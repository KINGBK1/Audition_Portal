# GLUG Audition Portal

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

First, clone the repository:

```bash
git clone https://github.com/lugnitdgp/Audition_Portal_Frontend.git
cd audition_portal_frontend
```

Next, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.

## Contributing

To contribute, please follow these steps:

1. Create a new branch for your feature or bug fix from the `development` branch:
   ```bash
   git checkout -b feature/your-feature-name development
   ```

2. Make your changes.

3. Commit your changes with a descriptive message:
   ```bash
   git add .
   git commit -m "Add feature: your feature name"
   ```

4. Push your branch:
   ```bash
   git push origin feature/your-feature-name
   ```

5. Create a Pull Request from your feature branch to the `development` branch.

6. Ensure your Pull Request is reviewed and approved before merging.
